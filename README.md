# Getting started with xCode, Node.js, React, MYSQL
Server:
- In VS Code terminal:
    > cd server
    > npm init -y 
    > npm i express mysql nodemon
    > add "type": "module", to package.json
    > add "start": "nodemon index.js" to scripts in package.json
    > npm i cors
    > npm i axios

Client:
- In VS Code terminal:
    > cd client
    > npx create-react-app .
    > npm start
    > npm i react-router-dom
    > npm i axios
    > npm i sass
